AMD Athlon com uma mistura de Pentium 4

Sim isso mesmo que você entendeu! Usei os dois processadores da época meu primeiro computador foi um Pentium 4 willamette uma das variantes com o clock mais baixo do processador.
Digamos que era um processador com um baixo desempenho para o que se esperava da Intel este pedaço de silicone conseguia ser ultrapassado pelo lendario Pentium 3 com núcleo tualatin que tinha clocks inferiores mas uma arquitetura muito superior.

Com o sistema tão potente como uma batata e uma placa-mãe com o famigerado chipset SiS(Sério ninguém merece) este sistema com incriveis 256MB de RAM mal conseguia rodar o Windows XP sem engasgar,culpa do o SO que possuia um overhead maior graças ao novo Kernel NT e culpa do hardware em geral.

Um núcleo de eficacia baixa não é uma boa receita para um computador de escritorio... Por tanto minha familia vendeu o sistema e montou um customizado um AMD Athlon ThunderBird(Tag Strong para um processador forte hihihi) acabamos com a variação com clock maior incriveis 1.9GHZ para a época ao contrario do anemico 1.4GHZ que o Pentium 4 oferecia.
A diferença? O canal PhilsComputerLab demonstra.
